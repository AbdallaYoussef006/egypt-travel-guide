# 🌍 Egypt Travel Guide

A **multi-page travel guide website** for Egypt covering top destinations, local parks, malls, restaurants, clubs, and transportation — built with HTML, CSS & JavaScript as a 1st-semester CS project at Nile University.

---

## ✨ Features

- 🏛️ **Top Destinations** — Cairo, Alexandria, Luxor, Aswan with descriptions
- 🌳 **Local Parks** — Al Azhar Park, Family Park, Zed Park, Africa Safari Park
- 🛍️ **Malls** — Mall of Egypt, City Stars, Cairo Festival City, Mall of Arabia
- 🍽️ **Restaurants** — Local Egyptian dining recommendations
- 🎉 **Clubs** — Entertainment and nightlife guide
- 🚗 **Transportation** — How to get around Egypt
- 📬 **Contact Form** — Get in touch section
- 🪐 **Exoplanet Page** — Bonus interactive science page with JavaScript

## 🚀 How to Run

**Requirements:** Any web browser

```bash
# Clone the repository
git clone https://github.com/AbdallaYoussef006/egypt-travel-guide.git

# Navigate to project
cd egypt-travel-guide

# Open in browser
open index.html
```
Or simply double-click `index.html` to open in your browser.
